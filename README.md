# 测试项目 TestProject

## 项目概述
TestProject 是一个用于测试 deepwiki 和 Ollama 解析能力的示例项目。  
该项目包含前端、后端、数据处理模块，以及示例模型集成，适合用于生成 Wiki、文档分析、自动化测试等场景。

主要功能：
- 演示项目结构解析
- 演示 README 中文解析
- 支持多模块、多文件结构
- 提供示例数据和测试脚本

---

## 系统架构
系统架构包含前端、后端和模型集成模块：
- **前端**：React + TypeScript
- **后端**：FastAPI + Python
- **数据库**：PostgreSQL
- **AI 模型集成**：PyTorch + Ollama
- **部署**：Docker + Docker Compose
